# README

## 仓库说明

个人算法和数据结构的学习笔记仓库.
其中包含 3 部分

1. resource 学习资料
2. notes 学习笔记
3. homework 家庭作业

## 目标

1. 核心
   1. 对于算法和数据结构有比较清楚的认识
   2. 能够支撑起一个Senior等级程序员的日常工作
2. 解题
   1. 完成 leetcode 必过的 200 题
   2. 能在 30 分钟内解出 medium 难度
3. 笔记整理
   1. 整理出个人能用的数据结构和算法笔记
   2. 常用数据结构,算法的演示代码

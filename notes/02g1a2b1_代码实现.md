# Linked list

In JavaScript, a linked list looks like this:

```js
const list = {
  head: {
    value: 6,
    next: {
      value: 10,
      next: {
        value: 12,
        next: {
          value: 3,
          next: null
        }
      }
    }
  }
}
};

```

linked list 基本实现的功能
- 遍历
- 新增
	在 linked-list 的尾部增加一个节点
- 指定位置，插入节点
- 指定位置，删除节点
